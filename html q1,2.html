sim <- function(p, sd){
  #Generate the first 10 coordinates of each µj vector j = 1, .., 3.
  m1 <- rnorm(10, mean=0, sd=1)
  m2 <- rnorm(10, mean=0, sd=1)
  m3 <- rnorm(10, mean=0, sd=1)
      X = matrix(rnorm(100*p,mean=0,sd=sd), 100, p) 
      for (col in 1:10){ 
        #group A
        for (a in 1:20){ 
        X[a,col] = rnorm(1, mean=m1[col], sd=sd)}
        #group B
      for (b in 21:50){ 
        X[b,col] = rnorm(1, mean=m2[col], sd=sd)}
        #group C
      for (c in 51:100){ 
        X[c,col] = rnorm(1, mean=m3[col], sd=sd)}}
      return(X)
}

Similarity Score: We chose the ‘Cophonetic distance’ method to compute the score. The value is the correlation similarity between two groups, and the range of it is between -1 and 1,with 0 meaning that the trees are not statisticall similar.

the Value we got is aprox 0.56 what shows that there is some statistical similar between the tree.

Finding the background distribution: In the quetion we permutate the names of the cities in the knesset dendogram and left cbs dendogram as it was. we did 300 iterations of permutation and saved the similarity score of each iteration using the same method before. When we got all the scores of the permutations estimated the density distribution of this data. Later we compared and calculated the p-value - we got pval 0.

The hypothesis of this simulation: H0: there is no correlation between the knesset and cbs datasets. H1: there is a correlation between the datasets.
